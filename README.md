# PoderGen — Generador de Poderes Notariales Bilingüe

Generador de poderes notariales bilingüe (ES/EN) para transacciones inmobiliarias en México.

**URL:** [podergen.expatadvisormx.com](https://podergen.expatadvisormx.com)

## Producto hermano de OfertaGen

- [OfertaGen](https://ofertagen.expatadvisormx.com) — Generador de ofertas de compraventa
- [Expat Advisor MX](https://expatadvisormx.com) — Asesoría legal para extranjeros

## Stack

- Next.js 14 + TypeScript + Tailwind CSS
- docx (generación DOCX lado cliente)
- Deploy: Vercel
- Repo: github.com/Pvrolomx/podergen

## Features

- Wizard 4 pasos: Partes → Inmueble → Facultades → Generar
- Documento bilingüe ES|EN en columnas paralelas
- 12 facultades configurables via checkboxes
- Artículos legales integrados (CCF + Código Civil Nayarit)
- PWA instalable
